PART 1

Project Overview & Brand Identity

#1.1 What ROKI Is

ROKI is a multi-layered Web3 ecosystem built around a minimalist white rabbit character with martial arts attitude. At its core sits ROKI Clash, a strategic combat game. Around it extends a network of companion products (ROKI Zen, ROKI Dojo, ROKI Stare, ROKI Dodge, ROKI Breath, ROKI Train, ROKI Verse, the NFT Launchpad, the NFT Generator, the AR-Lite Photo Maker, and future expansions like ROKI Lands) designed to reinforce one another and build a lasting character IP.

ROKI is not a single game with a token attached. It is a franchise strategy: the NFTs are equity-like stakes in an evolving universe, the token is the protocol currency, the games and minigames are engagement surfaces, and the launchpad is infrastructure that converts other projects' launches into buyback pressure for $ROKI. Each layer reinforces the others.

#1.2 Core Assets

  • 3,333 NFTs on Solana — unique character illustrations, 1:1 square format, minted via Metaplex standard
  • $ROKI token on Solana (pump.fun launch) — the ecosystem's native currency and burn target
  • ROKI Clash — the flagship competitive combat game (in development)
  • Companion products — Zen, Dojo, Stare, Dodge, Breath, Train, Verse, the Launchpad, the NFT Generator, and the AR-Lite Photo Maker, each designed to reinforce Clash and the broader ecosystem

#1.3 Brand Identity

  • Visual tone: grumpy rabbit with attitude — minimalist cartoon with thick black outlines, expressive faces, martial arts iconography
  • Thematic tone: dark, competitive, fight-night aesthetic balanced with character warmth
  • Design palette: Black (#0a0a0f), Red (#E94560), Green (#44CC88), Gold (#F5A623), Purple (#9B59B6), Blue (#4EA8DE), Yellow, Orange, Pink
  • Philosophy: halal-conscious, skill-based, dedication-rewarded, dignity for all characters, no pay-to-win, no gambling mechanics, earned progression over purchased shortcuts

#1.4 Positioning

What ROKI is deliberately not:

  • Not a generic PFP project with speculation as the only value
  • Not a pay-to-win game where wallet size decides the ladder
  • Not a gambling-adjacent token with loot boxes or manufactured FOMO
  • Not a single-product project that dies when one game loses attention

What ROKI is instead:

  • A character franchise in the making — multiple products reinforcing a single IP
  • A competitive skill-based ecosystem where reflex, strategy, and dedication are rewarded separately and distinctly
  • A living narrative universe where NFT holders are co-authors of the lore through their in-game actions
  • A protocol whose token is tied to real platform usage through the launchpad burn mechanism
