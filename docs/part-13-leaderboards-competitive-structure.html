PART 13

Leaderboards & Competitive Structure

#13.1 Multiple Leaderboards Design Principle

Single global leaderboards create the ‘same 5 winners every week’ problem. ROKI runs multiple parallel leaderboards so every player has something specific to chase.

#13.2 The Full Leaderboard Roster

Global Leaderboards

  • Global Clash ELO
  • Global Zen leaderboard (record-based, skill)
  • Global Stare leaderboard (longest hold)
  • Global Dodge leaderboard (survival time)
  • Global Breath leaderboard (longest daily streak)
  • Global Train leaderboard (longest daily streak)

Clan Leaderboards (5): Top Boxer / Tank / Grappler / Counter Puncher / Kicker

Elite Leaderboards

  • Top Copy Ninja
  • Top Mirror Fighter
  • Mirror vs Mirror sub-leaderboard (purest skill test — same style matchups by construction in v3)

Clan Wars Leaderboard — which clan is winning collectively this week/month/season

Head-to-Head Rivalry Records

#13.3 Rewards Are Discretionary, Not Scheduled

Leaderboards exist permanently. Rewards do not. The project awards prizes to top performers when budget permits. This prevents financial overcommitment and leverages variable-reward psychology.

#13.4 Intrinsic Value of Leaderboard Rankings

  • Public display on ROKI website homepage
  • Permanent achievement badges written to NFT metadata
  • Tournament qualification
  • Community recognition via ROKI Verse lore updates
  • Title inheritance (NFT carries titles forever, even after resale)

#13.5 Recognition Tiers for Free/Paid Players

  • Top 1000: Contender badge
  • Top 500: Rising badge
  • Top 100: Elite Contender badge
  • Top 16: Offered NFT promotion (see Part 14)

#13.6 Head-to-Head Rivalries

Every ROKI profile displays a Rivalries section showing opponents faced 3+ times, win/loss record, last match result, and current streak. AI flags rivalries at 3+ matches and generates rivalry paragraphs in the ROKI Verse.
